# ProjetoAgrinho2026
Este Site foi desenvolvido como parte do Projeto Agrinho,com o obejtivo de apresentar a importância e as características do agro no sudoeste do Paraná, destacando a agricultura, a sustentabilidade, a preservação ambiental e as tecnologias utilizadas no campo de forma criativa e interativa.

O projeto também serve como prática de programação web utilizando linguagem HTML, CSS e JavaScript.

# Funcionalidades
- Layout moderno e responsivo;
- Galeria de imagens do agro;
- Cards informativos;
- Navegação suave;
- Menu de acessibilidade;
- Efeitos visuais;
- Quiz interativo.
